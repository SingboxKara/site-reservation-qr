Bon à savoir

Fonctionnement

Les cartes fonctionnent par place. Si une réservation est faite pour 4 personnes, 4 places sont automatiquement utilisées sur la carte.

Utilisation

Le système a été pensé pour rester lisible : une seule carte selon vos habitudes, puis une déduction simple selon le nombre de participants sélectionnés.

Important

Les cartes ne devront pas être cumulables avec un code promotionnel au moment de leur utilisation. C’est une règle à bloquer côté backend.

Questions fréquentes

Les cartes sont-elles valables pour plusieurs personnes ?

Oui. Les cartes sont pensées pour fonctionner par place. Si vous venez à plusieurs, le nombre de places déduites correspond simplement au nombre de participants.

Peut-on utiliser une carte avec un code promo ?

Non. Pour garder un système cohérent et éviter les abus, l’utilisation d’une carte ne doit pas être cumulable avec un code promotionnel.

Pourquoi la carte après 15h n’est-elle pas valable le week-end ?

Parce que les créneaux week-end sont les plus demandés. Cette carte est volontairement limitée à la semaine pour rester attractive sans déséquilibrer la rentabilité.
