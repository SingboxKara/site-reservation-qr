Questions fréquentes sur les cartes Singbox

Comment fonctionne une carte Singbox ?

Une carte fonctionne par place. Une place correspond à une personne sur une réservation. Si vous venez à 5, 5 places sont déduites.

Peut-on utiliser une carte avec un code promo ?

Non. Les cartes Singbox ne sont pas cumulables avec un code promotionnel, afin de garder un système simple et cohérent.

Quand utiliser la carte après 15h ?

La carte après 15h est valable en semaine uniquement, hors vendredi soir et hors week-end. Elle a été pensée pour protéger les créneaux les plus demandés.

Où retrouver mes cartes achetées ?

Une fois achetées, elles apparaissent dans votre espace client, dans la page Mon compte.
